Connect
Setup guide

Install ttyd on your server, then run:

ttyd -W -p 7681 bash

Then connect to ws://yourserver:7681/ws

For SSH to another host:

ttyd -W -p 7681 ssh user@targethost

Tip: Add -t titleFixed=myserver to set a window title. Use -c user:pass for basic auth.

Run a WebSocket-to-TCP bridge on your server:

websockify 2222 localhost:22

Then connect to ws://yourserver:2222

Enter SSH target details below — they're sent as JSON on connect.

Deploy webssh2 on your server:

npm install -g webssh2
webssh2 --port 2222

Then connect to ws://yourserver:2222/ssh/host/TARGET

Argus sends SSH credentials via the proxy's protocol.

  2. Search for and enable the Google Drive API under APIs & Services → Library
  3. Go to OAuth consent screen → choose External → fill App name, User support email, Developer email → Save
  4. Under Test users, click "Add users" and add your own Google email address (required while app is in "Testing" mode)
  5. Go to Credentials → Create Credentials → OAuth client ID → Application type: Web application
  6. CRITICAL: Under "Authorized redirect URIs", click "Add URI" and paste the exact redirect URI shown below — must match character-for-character

Paste Providers

Push analysis reports, KG exports, and findings to paste services for sharing or archiving.

Same PAT as GitHub repo provider works here — needs gist scope
Setup Guide
  1. Go to GitHub Settings → Developer Settings → Personal Access Tokens
  2. Generate a token with the gist scope (or use your existing repo PAT if it has gist scope)
  3. Pastes are created as secret gists (unlisted, not indexed, but accessible via URL)
Setup Guide
  1. Go to pastebin.com/doc_api (free account required)
  2. Copy your Unique Developer API Key from the top of the page
  3. Username + password are optional — without them, pastes are created as guest (unlisted only)
  4. With login, you can create private pastes visible only to your account
Self-hosted PrivateBin instance — zero-knowledge encrypted, server never sees plaintext
Setup Guide
  1. Deploy PrivateBin on your own server (github.com/PrivateBin/PrivateBin)
  2. Or use a public instance: privatebin.net, paste.debian.net
  3. Enter the server URL above — no account or API key needed
  4. All encryption happens client-side — the server never sees your data

Keyword Routes Auto-file

Automatically route matching feed entries to projects. When new entries arrive, Argus scans headlines and descriptions for your keywords. Matches are flagged and auto-added to the target project.

keyword = include (OR) — matches if entry contains any keyword.
!keyword = exclude — rejects entry if that word appears, even if other keywords match.
Example: trump, biden, !tariff = entries with "trump" or "biden", but never "tariff".

About

Argus is an AI-powered web analysis toolkit with OSINT capabilities. Supports Grok, OpenAI, Claude, Gemini, and any OpenAI-compatible endpoint. Features streaming responses, custom editable prompts, multi-provider comparison, page monitoring, RSS feeds, smart bookmarks, project-based investigations, entity extraction, knowledge graph, source-aware pipelines, project dashboards with auto-digests, and more. Your API keys are stored locally and never shared.

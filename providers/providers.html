Setup Guide
  1. XMPP Server: Enter the WebSocket URL of your XMPP server (Prosody, ejabberd, Openfire). Usually wss://server:5281/ws or wss://server/xmpp-websocket
  2. JID: Your Jabber ID, e.g. nick@jabber.example.com
  3. Password: Your XMPP account password (encrypted via Vault)
  4. SMS Gateway: The domain of your SMS-over-XMPP gateway provider:
    • JMP.chat / Cheogram: cheogram.com — hosted service (~$4.99/mo, sign up at jmp.chat)
    • sms-over-xmpp: Self-hosted Go binary with GSM modem or Twilio backend
    • SMStoXMPP: Self-hosted PHP gateway using Android phone as SMS bridge
  5. Click Test Connection to verify your credentials and gateway reachability
How it works: Argus connects directly to your XMPP server via WebSocket. Messages to phone numbers are routed as XMPP stanzas to +number@gateway, where the gateway delivers them as SMS. No Argus servers involved.

Intelligence Providers

Connect OSINT and intelligence data sources. OpenSanctions and SEC EDGAR work without credentials. Others require API keys.

A free API key is required. Get one at opensanctions.org/api/ ↗. Provides sanctions lists, PEP databases, and entity screening across 30+ datasets.

SEC EDGAR requires no credentials. Provides corporate filings, insider transactions, and company facts for all US public companies.

All requests include a User-Agent header as required by SEC policy.

Free account required. Sign up at courtlistener.com/sign-in/ ↗, then get your token at courtlistener.com/profile/api-token/ ↗. Searches US federal and state court opinions, dockets, and citations.

Free subscription key required. Sign up at developer.trade.gov ↗, subscribe to "Data Services Platform APIs", then copy your key from your profile. Covers SDN, Entity List, Denied Persons, and 10 more U.S. sanctions lists.

EU Consolidated Financial Sanctions List requires no credentials. The full list is downloaded from the European Commission and searched locally. Updated within one working day of new Official Journal publications.

Free API key required. Register at sam.gov ↗, then request a public API key from your profile. Searches U.S. federal entity registrations by name, UEI, or CAGE code. Note: rate limited to ~1,000 requests/day.

Free API key required. Request at patentsview.org/apis/keyrequest ↗. Searches U.S. patent data — titles, inventors, assignees, CPC classifications. Rate limit: 45 requests/minute.

Free academic access available. Apply at lens.org/subscriptions ↗. Searches global patents across 100+ jurisdictions plus scholarly works. Free tier: 500 requests/day, 50,000 records/month.

AI-powered semantic patent search. Works without a token for basic use. For higher limits, email sam@projectpq.ai for a free academic/non-commercial token. Finds prior art using natural language queries.

Get credentials at Copernicus Dashboard ↗. Create an OAuth Client for Client ID + Secret. Instance ID is from Configuration Utility (needed for WMS tile layers).

Login to RadioReference.com → Account → Premium Subscription → copy your Service Key. Your $30/yr premium subscription unlocks the SOAP API used for county-level feed lookup.

FlightAware AeroAPI provides flight tracking, history, and airport data. Search by tail number, flight ID, or airline. Get your key at flightaware.com/aeroapi ↗

VesselFinder provides live vessel positions, details, and port calls. Optional — the ITU Ship Station database works for free vessel lookup without this key.

WiGLE maps WiFi, Bluetooth, and cellular networks by location. Create a free account at wigle.net ↗, then go to Profile → API Token and copy your API Name (starts with AID) and API Token. Used to overlay detected wireless networks on the satellite and geomap views.

Stadia Maps provides premium map tiles including Stamen styles (Toner, Watercolor, Terrain), Alidade Satellite, and Outdoors. Sign up at client.stadiamaps.com ↗, create a property, then generate an API key under Authentication. Used as basemap tile providers on the satellite page.

Windy Webcams API v3 provides access to thousands of live webcams worldwide, searchable by location. Sign up at api.windy.com ↗ and create a Webcams API key. Used to overlay live camera locations on the satellite map with thumbnail previews and live stream links.

Windy provides two separate API keys. The Map Tiles key powers the weather tile overlay on the satellite map (wind, rain, temperature, etc). The Point Forecast key powers the right-click weather popup with 48h forecasts from GFS/ECMWF/ICON. Both are optional independently. Get keys at api.windy.com ↗.

OpenWeatherMap provides weather tile overlays (rain, wind, temperature, clouds, pressure) and current conditions at any lat/lon. Get a free API key at home.openweathermap.org/api_keys ↗. Without a key, current conditions fall back to Open-Meteo (free, no key needed) but tile overlays require the OWM key.

Department of Labor Data Portal. Currently provides MSHA mine safety data (accidents, mine addresses). OSHA inspections will be available when DOL publishes them to the v4 API. Register for a free key at dataportal.dol.gov/registration ↗.

Federal Election Commission campaign finance data — political donations, PAC spending, candidate committees. Get a free API key at api.open.fec.gov/developers ↗.

ProPublica Nonprofit Explorer — 990 filings for every US nonprofit, including revenue, expenses, officer salaries. Get a free API key at propublica.org/datastore/api ↗.

Paste Providers

Push analysis reports, KG exports, and findings to paste services for sharing or archiving.

Same PAT as GitHub repo provider works here — needs gist scope
Setup Guide
  1. Go to GitHub Settings → Developer Settings → Personal Access Tokens
  2. Generate a token with the gist scope (or use your existing repo PAT if it has gist scope)
  3. Pastes are created as secret gists (unlisted, not indexed, but accessible via URL)
Setup Guide
  1. Go to pastebin.com/doc_api (free account required)
  2. Copy your Unique Developer API Key from the top of the page
  3. Username + password are optional — without them, pastes are created as guest (unlisted only)
  4. With login, you can create private pastes visible only to your account
Self-hosted PrivateBin instance — zero-knowledge encrypted, server never sees plaintext
Setup Guide
  1. Deploy PrivateBin on your own server (github.com/PrivateBin/PrivateBin)
  2. Or use a public instance: privatebin.net, paste.debian.net
  3. Enter the server URL above — no account or API key needed
  4. All encryption happens client-side — the server never sees your data
